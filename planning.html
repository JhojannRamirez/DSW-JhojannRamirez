Estructura básica de HTML5 (Etiquetas, atributos y valores)

1. Objetivos del Sitio

1.1 Objetivo General

Desarrollar una Estrucutra de Datos, entre otros sistemas, para conocer y aprender sobre las tematicas de aprendizaje de la: Estructura basica de datos HTML 5 (etiquetas, atributos y valores) para la creacion de un OVI que solicitado por la UNAD para la empresa htmlDesing que la cual nos a sido contratada como Diseñadores Web .

1.2 Objetivo Especificos

I. Establecer estrategias de aprendizaje virtuales de la tematica de Diseños de Sitios Web mediante la estrategia de desarrollo de Objetos Virtuales(OVA) y Ojetos Virtuales de Informacion(OVI).

II. Desarrollar un proyecto OVI para ser presentado al cliente UNAD y la esquematizacion que tendra lugar en un Sitio Web creado a a partir de una estructura de base de datos HTML5 y otros componentes.

III. Presentar mediante un borrador el proyecto para que sea visualizado y evaluado por la pagina y en esta misma la maquetacion del Sitio(Mockup), y asi mismo ser sincronizado en el repositorio para que la pagina web sea evaluada.

En el OVI, se subdividirá en varias secciones para explicar la tematica relacionada con Estructura basica de datos html5:(etiquetas, atributos y valores) y selectores CSS Las cuales se destacará las siguientes para brindar una excelente experiencia al usuario final:
